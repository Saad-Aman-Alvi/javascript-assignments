JavaScript Expression Evaluation

Expression: var a = 2, b = 1; var result = --a - --b + ++b + b--;

Step-by-Step Explanation:

  • Initial values: a = 2, b = 1
  • --a: Pre-decrement → a becomes 1
  • --b: Pre-decrement → b becomes 0
  • ++b: Pre-increment → b becomes 1
  • b--: Post-decrement → Value used is 1, then b becomes 0
  • Expression: --a - --b + ++b + b-- becomes 1 - 0 + 1 + 1 = 3
